# Games Website

Gaming themed website built with HTML, CSS and JavaScript.

## Features

- Responsive layout
- Modern gaming design
- Interactive UI elements
- Download section
- Multi-page structure

## Technologies

- HTML5
- CSS3
- JavaScript

## Project Structure

```bash
css/              # styles
img/              # images and assets
js/               # scripts
download-files/   # downloadable content
```
## Pages
* Home page
* Download page
* About

This project was created to practice frontend web development and improve skills in building responsive websites using vanilla HTML, CSS and JavaScript.
